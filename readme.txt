== Installation ==

1. Upload the `wpmatch` folder to `/wp-content/plugins/`
2. Activate the plugin through 'Plugins' menu in WordPress
3. WooCommerce will be automatically installed if not present (WordPress 6.5+)
4. Navigate to WPMatch > Settings to configure
5. Use WPMatch > Membership Setup to create membership products
6. Add shortcodes to your pages

= Shortcodes =
* `[wpmatch_search]` - Display the search interface
* `[wpmatch_matches]` - Show user matches
* `[wpmatch_premium_shop]` - Display membership options
